## MonoBrancher

# Project name: Monolingual Brancher
This application takes a Japanese String and splits it using the Mecab library.
The use is able to click on any word in that sentence, and will be provided with the Definition of that word.
If the user does not know a word in the definition, clicking that word will show the new definition, and add that word
to the Tree that can be traced back to comprehend the path that was taken to comprehend the initial word.
